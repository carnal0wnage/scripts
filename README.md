scripts
=======
Scripts to automate various INFOSEC related tasks.

# Pentesting Stuff #
* Scripter.rb - ensures that our xterms are scripted and logging output for accountability
* printer.rb - feed it an HP printer IP, it checks if we can see the domain password
* survey.rb - some automated post-exploitation stuff for meterpreter
* x11grab.rb - automatically grab screencaps from insecure X11 servers

# DFIR Stuff #
* analyzeVol.pl - some automated stuff for volatility output
* runVol.pl - automate running a bunch of volatility plugins

# Other Random #
